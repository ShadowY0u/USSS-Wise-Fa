Procédure

Règles d’intervention et de fonctionnement du U.S.S.S.

Standard opérationnel

Schéma d’intervention (simple)

  • 1. Observation : identifier la situation, sécuriser la zone.
  • 2. Contact : annonce claire, posture professionnelle, contrôle.
  • 3. Décision : appliquer la procédure adaptée (contrôle, escorte, neutralisation).
  • 4. Compte-rendu : rapport écrit + preuves si nécessaire.
Voir les divisions → Règles supervision →
Contrôle

Contrôle d’identité

  • Motif + annonce de contrôle.
  • Fouille / palpation selon contexte (sécurité).
  • Vérification informations + décision (libération / escorte / retenue).
Sécurité

Gestion périmètre

  • Définir zone : interne / externe / accès.
  • Placer points de contrôle et consignes radio.
  • Autorisation entrée : liste / badge / validation supervisor.
Radio

Codes & communication

  • Messages courts, clairs, sans spam.
  • Priorité : sécurité, localisation, besoin renfort.
  • Fin d’action : “situation sous contrôle” + résumé.
Rapport

Compte-rendu obligatoire

  • Qui / quoi / où / quand.
  • Mesures prises + justification.
  • Pièces : captures, vidéo, témoins.
